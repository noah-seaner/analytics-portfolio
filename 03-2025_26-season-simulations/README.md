# Conference Season Simulation Framework

## Project Overview

This project consists of a series of simulation models designed to forecast conference performance and Patriot League tournament outcomes throughout Lafayette's 2025-26 campaign. Rather than relying on a single static model, the simulations were updated and expanded as the season progressed, incorporating new game data to refine projections.

The framework includes simulations of Lafayette's conference play based on their non-conference performance, iterative updates as Lafayette played more conference games, and a separate simulation of the conference tournament bracket. Together, these models provided a probabilistic view of potential outcomes across different stages of the season.

## Objective

To estimate possible conference records and tournament outcomes using team performance metrics and simulation-based forecasting, while updating projections dynamically as new games were played.
