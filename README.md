# Data Analytics & Sports Analytics Portfolio - Noah Seaner

This repository contains selected analytical projects demonstrating experience in statistical modeling, data visualization, and applied data analysis across sports and business domains. Projects utilize R, Excel, and Tableau to extract insights from large and complex datasets and communicate results to stakeholders.

## Featured Projects

### Grammy.com Website Analysis (Excel)
Conducted exploratory and statistical analysis of website traffic and engagement metrics to identify growth opportunities and inform A/B test-driven recommendations.

### Intel Data Center Site Selection Dashboard (Tableau)
Built interactive dashboards to analyze energy availablity and consumption trends across potential data center locations. Visualizations supported comparative evaluation of site feasibility and operational sustainability.

### Conference Record Prediction & Simulation Model (R)
Developed a predictive modeling framework to estimate conference win outcomes using game-level performance data. Implemented regression-based modeling and simulation techniques to forecast season record distributions and evaluate performance scenarios.

### Bayesian Analysis of Torpedo Bats in MLB (R)
Applied Bayesian statistical methods to evaluate the impact of the short-lived torpedo bat on offensive performance. Modeled uncertainty and effect size to assess whether observed performance changes were statistically meaningful.
