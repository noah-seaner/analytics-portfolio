# Analytics Portfolio - Noah Seaner

Welcome to my analytics portfolio! This repository highlights a selection of projects that reflect my interests in data analysis, statistical modeling, and sports analytics. Across these projects, I work primarily with Excel, Tableau, and R to explore datasets, build models, and create dashboards that turn raw data into actionable insights.

While some projects focus on sports performance and predictive modeling, others explore business and operational analytics across different industries. Together, they showcase how I approach analytical problems: start with a clear question, choose the right tools, and communicate results in a way that supports decision-making.

### Grammy.com Website Analysis (Excel)
This project analyzes website traffic and engagement metrics to identify growth opportunities for grammy.com. Using Excel for exploratory analysis and visualization, I evaluated user trends and supported A/B test-driven recommendations aimed at improving audience engagement.

### Intel Data Center Site Selection Dashboard (Tableau)
This Tableau project analyzes energy availability and consumption trends to evaluate potential locations for a new data center. The interactive dashboards compare regional energy profiles and highlight trade-offs between capacity, stability, and operational feasibility.

### Conference Record Prediction & Simulation Model (R)
This project focuses on predicting outcomes of conference games using game-level performance data. I developed regression-based models and simulation methods to estimate season win distributions and evaluate how team performance in non-conference play translates to conference results. The project is still evolving as more data is added, reflecting an iterative modeling process.

### Bayesian Analysis of Torpedo Bats in MLB (R)
In this analysis, I used Bayesian statistical methods to examine the impact of the torpedo bats on offensive production in Major League Baseball. The goal was to assess whether the observed performance spike was meaningful or just a temporary fluctuation, while explicitly modeling uncertainty in the estimates.

All of these projects reflect my interest in combining statistical modeling with real-world decision contexts, particularly in sports and performance analytics.
