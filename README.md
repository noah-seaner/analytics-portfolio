# Analytics Portfolio - Noah Seaner

Welcome to my analytics portfolio! This repository contains a curated selection of projects that demonstrate my work in data analysis, statistical modeling, and sports analytics.

Across these projects, I explore how data can be used to answer real-world questions through simulation modeling, statistical analysis, and data visualization. The projects span multiple domains, including business analytics, infrastructure planning, and sports performance analysis. 

## Tools Used
- **Excel** - data transformation, KPI analysis, and business analytics
- **Tableau** - interactive dashboards and data visualization
- **R** - simulation modeling and statistical analysis 

## Featured Projects

### Grammy Website Analytics & KPI Evaluation
A business analytics project using Excel to transform website traffic data into actionable KPIs for The Recording Academy's websites. The analysis includes trend evaluation, engagement benchmarking, competitor comparisons, and statistical testing to inform recommendations for improving website performance.

[View Project ->](./01-grammys-website-analytics)

### Intel Data Center Site Selection Dashboard
An interactive Tableau dashboard analyzing regional energy production and sustainability metrics to support infrastructure planning decisions. The dashboard compares net energy production, renewable energy share, and monthly generation trends across multiple regions.

[View Project ->](./02-intel-site-selection-dashboard)

### Conference Season Simulation Framework (R)
A season-long simulation framework used to forecast conference records and Patriot League tournament outcomes in college basketball. The project includes projections based on non-conference performance, iterative updates as conference games were played, and simulations of how the conference tournament bracket. These models provided a probabilistic view of how team performance and postseason scenarios evolved throughout the season.

[View Project ->](./03-2025_26-season-simulations)

### Bayesian Analysis of Torpedo Bats in MLB (R)
A Bayesian statistical analysis evaluating whether MLB players who adopted torpedo bats experienced measurable changes in offensive performance. The project compares batting statistics across seasons using hierarchical modeling and MCMC sampling to estimate performance effects while accounting for player-level variability.

[View Project ->](./04-bayesian-torpedo-bats-mlb)

## Project Organization
The repository is organized to highlight a small set of featured projects while preserving earlier work and iterations in an archive folder. Each featured project includes documentation explaining the objective, analytical approach, and key insights derived from the analysis.
